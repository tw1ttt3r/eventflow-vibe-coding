# Instrucciones para agentes

## Bitácora de interacciones

- Toda interacción con el usuario debe registrarse en `BITACORA.md` antes de finalizar el turno.
- El registro debe incluir la fecha, el prompt literal proporcionado por el usuario, un resumen breve de la solicitud, la respuesta o cambio realizado, y el estado de pruebas/comandos relevantes.
- Si se modifican archivos, la entrada de bitácora debe mencionar los archivos principales afectados.
- Esta instrucción aplica a todo el repositorio.
